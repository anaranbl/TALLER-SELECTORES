Selectores

  1. Selector de etiqueta

    2. Se aplica a todas las etiquetas HTML de un tipo específico.

  2. Selector de ID (#)

    3. Se aplica a un único elemento con un id específico.

  3. Selector de clase (.)

    4. Se aplica a todos los elementos con una clase específica.

  4. Selector múltiple (,)

    5. Aplica el mismo estilo a varios elementos.

  5. Selector de atributos ([])

    6. Aplica estilos a elementos con un atributo específico.

  6. Selector universal (*)

    7. Aplica estilos a todos los elementos de la página.

  7. Selector de descendencia (espacio)

    8. Selecciona elementos dentro de otro elemento (sin importar cuán profundo estén).

  8. Selector de hijo directo (>)

    9. Aplica estilos solo a elementos que son hijos directos de otro.

  9. Selector de hermano adyacente (+)

    10. Selecciona el primer elemento que sigue inmediatamente a otro.

  10. Selector de hermanos generales (~)

    11. Selecciona todos los elementos que son hermanos del mismo nivel después de otro.
